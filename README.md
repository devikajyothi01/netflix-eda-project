# 📊 Netflix Data Analysis Dashboard

## 📌 Project Overview
This project focuses on cleaning, analyzing, and visualizing the Netflix dataset to uncover meaningful insights about content distribution, trends, and ratings.

## 🧹 Data Cleaning
- Handled missing values (e.g., director, cast, country)
- Replaced null values with appropriate labels
- Removed inconsistencies and prepared dataset for analysis

## 📊 Data Visualization
Visualizations were created using:
- Python (Pandas, Matplotlib, Seaborn)
- Power BI Dashboard

Key visualizations include:
- Movies vs TV Shows distribution
- Content release trend over years
- Ratings distribution
- Genre analysis
- 📊 Visualizations

## 🔍 Key Insights
- Majority of content on Netflix are Movies
- Content production increased significantly after 2015
- TV-MA is the most common rating
- Drama and International genres are highly popular
